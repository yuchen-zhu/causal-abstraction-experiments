# Causal Abstraction Experiments

Numerical experiments accompanying Chapter 4.2 of the thesis:
**"Constructive Causal Abstraction: Deriving Consistent Interventions via Complementary Variables"**

## Experiments

| Script | Description |
|--------|-------------|
| `experiment1_linear_validation.py` | Validates Theorem 1: the `q = [pA - Āp, p]` construction renders all macro interventions well-defined in the linear case. |

## Setup

```bash
pip install -r requirements.txt
```

## Running

```bash
python experiment1_linear_validation.py
```
